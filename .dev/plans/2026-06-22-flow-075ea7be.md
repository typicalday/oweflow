## File to touch
- `.github/workflows/ci.yml` — the only file in scope.

## Exact changes

- Before: `      - uses: actions/checkout@v4`
- After:  `      - uses: actions/checkout@v5`

## Approach
Two targeted string replacements in `.github/workflows/ci.yml`. No logic change; purely cosmetic version pin bumps.
